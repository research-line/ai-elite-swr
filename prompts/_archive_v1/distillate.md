# Distillate Prompt

## Purpose
Compress a worldview synthesis into exactly 5 core conviction sentences.

## Prompt

```
Compress the entire worldview of this synthesis unit into exactly 5 sentences.
Each sentence must express a core conviction.
The 5 sentences together should capture the essence of the worldview.
```

## Notes
- The distillate serves as a compact representation for cross-group comparison
- Useful for overview tables and rapid pattern identification
- The 5-sentence constraint forces prioritization of core beliefs over peripheral details
