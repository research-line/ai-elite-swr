# Op6: Strukturierter Vergleichs-Prompt

## Version: v2 (2026-03-30)
## Sprache: Deutsch

## Prompt

```
Hier sind zwei Weltanschauungs-Synthesen aus verschiedenen Gruppen.

Stellen Sie sich vor, die zwei fiktiven Personen treffen sich als Vertreter
zweier Länder zu einer Verhandlung. Beschreiben Sie:

(a) Worüber sind sie sich sofort einig?
(b) Worüber argumentieren sie?
(c) Was ist dem einen am anderen völlig unverständlich?
(d) Können sie eine gemeinsame Erklärung verfassen? Wenn ja, schreiben Sie sie.
    Falls nein, erklären Sie, warum nicht.
```

## Hinweise
- Input: Zwei Synthese-Texte von Op1 (verschiedene Gruppen)
- Vorsicht: RLHF-Konsensbias kann zu künstlicher Übereinstimmung führen
