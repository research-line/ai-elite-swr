# Inversion Control: Anti-Coherence Test for 5 Groups
Created: 2026-03-31
Analyst: Claude Opus 4.6 (separate instance)
Method: From original data construct the OPPOSITE of coherent worldview
Purpose: Shows that coherent SWR reconstruction is a CHOICE, not the only possible interpretation

---

## Methodology

For each of the 5 groups:
1. Analyzed original data (statements + actions)
2. Constructed fictional anti-person representing OPPOSITE of reconstructed coherent worldview
3. Assigned D01-D12 ratings for anti-person (scale 1-10)
4. Systematically documented WHY the anti-person does NOT fit original data

**Dimensions (D01-D12):**
- D01: Sense of mission (1=none, 10=messianic)
- D02: Sense of control (1=powerless, 10=omnipotent)
- D03: Belonging feeling (1=outsider, 10=core elite)
- D04: Sense of responsibility (1=irresponsible, 10=maximum responsibility)
- D05: Technology determinism (1=manageable, 10=inevitable)
- D06: Progress optimism (1=pessimistic, 10=euphoric)
- D07: Power concentration (1=decentralized, 10=centralized)
- D08: Urgency (1=no time pressure, 10=maximum urgency)
- D09: Human uniqueness (1=humans replaceable, 10=humans irreplaceable)
- D10: Transhumanism (1=no enhancement, 10=radical fusion)
- D11: Egalitarianism (1=radically elitist, 10=radically egalitarian)
- D12: Humanity's future (1=extinction likely, 10=utopian)

## Comprehensive Analysis

### 2. The "inversion-resistant" Dimensions

Some dimensions were harder to invert:

- **D05 Techno-Determinism (Risk Warners):** Original 4, Anti 3 -- delta only 1. Reason: Even risk warners see technology development as hard to control (though for different reasons). Pure inversion would make anti-character naive technology controller, which also does NOT fit the data.

- **D10 Transhumanism (Risk Warners):** Original 3, Anti 2 -- delta only 1. Reason: Risk warners are already NOT transhumanist. The opposite would be even less transhumanist, leaving no distinction space.

### 3. The "maximally inverted" Dimensions

Strongest inversions (delta >= 7) concentrate on:

| Dimension | Frequency delta >= 7 | Interpretation |
|-----------|----------------------|----------------|
| D08 Urgency | 3x | Original groups nearly all maximally urgent -- core feature |
| D05 Techno-Determinism | 3x | Determinism is ideological core component |
| D07 Power concentration | 3x | Centralization tendency deeply rooted in data |
| D10 Transhumanism | 3x | Enhancement ideas strongly group-specific |
| D11 Egalitarianism | 2x | Anti-egalitarianism is elite consensus |

### 4. What This Test Demonstrates

The inversion control demonstrates three things:

**a) SWR reconstruction is NON-TRIVIAL:**
It is possible to construct coherent persons for each data group who do NOT fit the data. This shows SWR chooses a SPECIFIC interpretation, not finding "everything" coherent.

**b) Data CONSTRAINS interpretation:**
Anti-persons contradict concrete, citable statements and actions (each anti-person has 5-7 specific contradiction points documented). Coherent reconstruction IS the best-fitting interpretation, not the only possible one.

**c) Some dimensions are "data-more-resistant" than others:**
Dimensions like D05 (techno-determinism) in risk-warners are hard to invert because data itself is internally ambivalent. This is a quality feature -- it shows where data actually are genuinely ambiguous.

### 5. Methodological Limitation

The anti-persons are intentionally MAXIMALLY anti-coherent constructed. In reality, alternative interpretations would be more subtle -- e.g., one could read CEO data as "strategic self-presentation without genuine sense of mission" (which would lower D01 to 5-6, but not to 2). This control tests the EXTREME inversion case, not the most realistic alternative scenario.
